# 图片素材放置说明

如果后续使用 AI 生图或其他网站下载素材，请放到这个目录。

建议文件名：

- `app-bg.jpg`：全局温暖深空背景
- `planet-overview.jpg`：首页总览星球
- `hub-sport.jpg`：运动绿洲入口图
- `hub-diet.jpg`：饮食花园入口图
- `hub-study.jpg`：学习天文台入口图
- `hub-work.jpg`：工作基地入口图
- `hub-plan.jpg`：计划灯塔入口图
- `hub-sleep.jpg`：月光观测站入口图
- `ecology-sport.jpg`：运动绿洲详情图
- `ecology-diet.jpg`：饮食花园详情图
- `ecology-study.jpg`：学习天文台详情图
- `ecology-work.jpg`：工作基地详情图
- `ecology-plan.jpg`：计划灯塔详情图
- `ecology-sleep.jpg`：月光观测站详情图
- `space-garden-bg.png`：温暖星空背景
- `planet-card-lv1.png`：荒芜星展示图
- `planet-card-lv3.png`：绿洲星展示图
- `achievement-badges.png`：成就徽章素材合集
- `share-card-bg.png`：星球名片背景

可选动画文件名：

- `planet-idle.mp4`：首页星球轻微自转
- `starlight-burst.mp4`：打卡成功星光粒子
- `zone-glow.mp4`：生态区灯光呼吸

当前版本会优先使用压缩后的 JPG。原始 PNG 已备份到 `assets/images-original/`，并在 `project.config.json` 中排除打包，避免小程序本地包过大。

# 星途自律舱

**星球养成式自律打卡微信小程序** — 把运动、饮食、学习、工作、计划、睡眠六类日常行动，连接到一个会生长的星球生态。每一次打卡获得星光值，逐步点亮荒芜星球，让它从灰色岩石变成繁荣的生态家园。

---

## 项目特色

| 特色 | 说明 |
|------|------|
| 🪐 **成长星球** | 8 级星球体系（荒芜星→光环星），打卡累积星光值驱动星球从荒芜到繁荣，表面长出植被、河流、建筑和光环 |
| 🌍 **六维生态区** | 运动绿洲、饮食花园、学习天文台、工作基地、计划灯塔、月光观测站，每个维度有独立生态区和可视化成长反馈 |
| 🎯 **目标预设** | 5 种内置自律目标（均衡自律 / 学习提升 / 健身塑形 / 项目冲刺 / 早睡早起），选择后系统自动推荐今日行动 |
| 🚀 **行动舱** | 学习、工作、运动等维度进入沉浸行动舱，计时专注、记录成果，再结算星光和生态变化 |
| 📋 **计划中心** | 用户可以添加个性化计划，选择维度、行动内容和提醒时间，记录每天是否完成，支持编辑、复用和直接进入行动舱 |
| ⭐ **星光奖励体系** | 打卡后即时弹出奖励弹窗，显示星光结算、生态解锁内容和宝箱奖励，避免"做完没感知" |
| 🎁 **成就墙** | 7 种成就徽章（初次点亮 / 完美一天 / 七日航线 / 学习恒星 / 运动轨道 / 规律作息 / 星光收藏家），长期行动沉淀为可展示的荣誉 |
| 🔔 **提醒体系** | 「我的」页面内置打卡提醒，计划中心支持单个计划开始时间，帮助用户建立行动节奏 |
| 📊 **数据复盘** | 星光值、当前等级、连续天数、打卡总数、云端同步状态一目了然，热力图展示近期行动密度 |
| 🚀 **新手引导** | 首次启动播放星球动画视频，完成星球命名、目标选择，建立用户与星球之间的拥有感 |
| 🔁 **云端同步** | 接入微信云开发，支持登录、打卡提交、统计读取、资料更新、数据导出和云资源链接 |
| 🎨 **轻量包体** | 大图和视频走云存储，本地包内仅保留压缩后的关键资源，避免包体超限 |
| 🧪 **单元测试** | 核心计算逻辑（日期、目标、奖励、统计、成就、生态）使用 Node.js Test Runner 覆盖 |

## 六大生态区

| 生态区 | 维度 | 场景名 | 视觉反馈 |
|--------|------|--------|---------|
| 🏃 运动绿洲 | 运动 | 跑道、球场和围栏花园 | 起点跑道 → 晨跑花径 → 能量看台 → 训练营地 |
| 🥗 饮食花园 | 饮食 | 果园小屋和露台餐桌 | 补水井 → 果树篱笆 → 露台餐桌 → 丰收小屋 |
| 📚 学习天文台 | 学习 | 图书馆灯光和观测穹顶 | 阅读灯 → 书页花园 → 图书馆侧楼 → 观测穹顶 |
| ⚡ 工作基地 | 工作 | 办公楼、实验室和信号塔 | 启动工位 → 任务路径 → 实验室 → 信号塔 |
| 🎯 计划灯塔 | 计划 | 灯塔、航线牌和任务广场 | 方向牌 → 任务广场 → 灯塔光束 → 航线中心 |
| 🌙 月光观测站 | 睡眠 | 望远镜、月光花园和静眠舱 | 月光长椅 → 静眠花圃 → 观测台 → 梦境圆顶 |

每个生态区包含 4 级解锁体系（1 次 / 3 次 / 7 次 / 14 次打卡），进度可视化标注"还差 X 次解锁下一阶段"。

## 奖励计算规则

- 基础打卡：10 星光值
- 轻量 / 标准 / 深度 努力加权：0 / +4 / +10
- 记录笔记（6 字以上）：+2
- 完美一天（六项全完成）：+30 星光值
- 连续打卡 7 天倍数日：+80 星光值

## 打卡动作卡

每个维度提供 3 档行动卡片（轻量 / 标准 / 深度）：

| 维度 | 轻量 | 标准 | 深度 |
|------|------|------|------|
| 运动 | 微汗启动（15 分钟） | 绿洲建设（30 分钟） | 突破训练（45 分钟+） |
| 饮食 | 补水果园（饮水） | 三餐巡检（记录健康评分） | 健康餐盘（全天规律） |
| 学习 | 专注 25 分钟 | 知识推进（60 分钟） | 深度沉浸（90 分钟+） |
| 工作 | 清掉一个小任务 | 项目推进（1 小时） | 深度工作舱（2 小时） |
| 计划 | 写下三件事 | 任务复盘 | 航线规划（周计划） |
| 睡眠 | 睡前降噪 | 规律作息 | 早睡航线 |

## 云开发接入

项目已包含 6 个云函数目录，首次配置请参考 **[docs/cloud-setup.md](docs/cloud-setup.md)**。

**云函数清单**：

| 云函数 | 功能 | 触发方式 |
|--------|------|---------|
| `login` | 微信登录获取 openid | 小程序启动自动调用 |
| `submitCheckin` | 提交打卡记录到云端 | 打卡完成后调用 |
| `getDashboard` | 拉取云端统计汇总 | 进入数据页时调用 |
| `exportData` | 导出用户全部云端数据 | 我的页面手动触发 |
| `updateProfile` | 同步个人资料到云端 | 更新昵称/星球名后调用 |
| `getAssetUrl` | 获取云存储资源下载链接 | 加载图片视频时调用 |

> 提示：首次使用需在微信开发者工具中点击「云开发」→ 创建数据库集合（`checkins`、`profiles`），右键云函数目录选择「上传并部署」。

## 行动与计划闭环

1. 首页选择目标，明确当前阶段要养成的节奏。
2. 打卡页选择生态区，优先进入对应行动舱完成真实行动。
3. 计划中心保存长期计划，可设置开始时间、编辑内容、记录当天是否完成。
4. 完成后即时弹出星光结算、宝箱进度、生态解锁和成就反馈。
5. 数据页与生态区页面展示长期变化，让用户看到习惯积累的结果。

## 关于

**星途自律舱** 是一款面向大学生和职场新人的自律打卡工具，核心理念是 **"先让用户有目标，再让打卡有反馈"**。它不只是一个打卡清单，而是一个完整的成长体验：从目标设定、自定义计划、每日行动、即时奖励、生态变化到数据复盘，每一步都有可视化反馈。

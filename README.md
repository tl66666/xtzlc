# 星途自律舱

> 把每天的行动，养成一颗会生长的星球。

星途自律舱是一款 **星球养成式自律打卡微信小程序**。它不只是"点一下完成"，而是围绕 **制定计划 → 专注执行 → 完成打卡 → 即时奖励 → 星球成长 → 数据复盘** 构建了一套完整的行动反馈闭环，让自律变得有目标、有反馈、有成长感。

## 项目亮点

### 1. 专注模式 — 行动舱沉浸计时

用户制定计划后，可进入行动舱开始专注计时。行动舱提供全屏沉浸体验，覆盖学习、工作、运动等真实场景。计时结束后自动结算星光奖励并同步计划完成度，让"专注"不再是一个模糊的概念，而是一次有起止、有成果的行动。

### 2. 计划中心 — 制定计划，完成打卡

用户可以创建、编辑、删除自己的每日计划，为每条计划设置：
- **归属维度**（运动 / 饮食 / 学习 / 工作 / 计划 / 睡眠）
- **具体行动内容**（如"阅读 30 分钟"、"跑步 5 公里"）
- **目标量**（次数 / 时长 / 距离）
- **提醒时间**

计划完成后自动归档并计入星球成长，未完成的计划会在次日提醒，形成"计划 → 执行 → 反馈"的完整链路。

### 3. 每日提醒 — 不让任何一天空白

每个计划可以独立设置提醒时间。到点后微信推送通知，点击直接跳转到对应计划的行动舱入口。连续打卡天数会实时显示在首页，断签会清零，激励用户保持节奏。

### 4. 星球成长叙事

用户首次进入时完成星球启动动画和命名仪式。之后每次行动都会让星球从荒芜逐步复苏，经历 9 个成长阶段：荒芜 → 微光 → 苔藓 → 溪流 → 森林 → 城市 → 科技 → 星际文明。每个阶段都有独立的视觉状态和生态解锁。

### 5. 六类生态区

六类日常行动分别对应六个独特生态区：

| 维度 | 生态区 | 成长内容 |
| --- | --- | --- |
| 运动 | 运动绿洲 | 跑道、球场、健身花园 |
| 饮食 | 饮食花园 | 果园、小屋、健康餐桌 |
| 学习 | 学习天文台 | 图书馆、知识观测穹顶 |
| 工作 | 工作基地 | 办公楼、实验室、信号塔 |
| 计划 | 计划灯塔 | 航线牌、任务广场 |
| 睡眠 | 月光观测站 | 望远镜、月光花园 |

### 6. 即时奖励反馈

打卡完成后直接弹出组合反馈：

- **星光值**：每完成一项打卡获得 10-50 星光
- **生态点亮**：对应生态区出现新建筑或装饰
- **宝箱进度**：六项全部完成解锁每日宝箱
- **成就提示**：达成里程碑时弹出成就徽章

### 7. 数据复盘

数据页提供多维度的自我观察工具：

- 连续打卡天数
- 本周完成率
- 热力图日历
- 六维趋势曲线
- 成就徽章墙
- 数据导出（JSON）

### 8. 数据可控

昵称、星球名、目标、计划、提醒、本地数据均提供编辑或清理入口，用户可以随时重新定制自己的使用节奏。

## 核心流程

```text
启动星球 → 命名星球 → 选择近期目标
  ↓
创建每日计划（设置维度 / 目标量 / 提醒时间）
  ↓
到点提醒 → 进入行动舱专注计时
  ↓
完成打卡 → 即时结算（星光 + 宝箱 + 生态 + 成就）
  ↓
星球成长 → 生态区解锁 → 数据沉淀 → 次日继续
```

## 功能模块

| 模块 | 说明 |
| --- | --- |
| 星球总览 | 首页展示星球名称、日期、星光、今日进度、目标状态、生态入口和下一步行动 |
| 目标体系 | 支持选择和清除近期目标，目标会影响首页推荐、打卡模板和行动优先级 |
| 计划中心 | 创建、编辑、删除计划，设置类别、行动内容、目标量、提醒时间和备注 |
| 行动舱（专注模式） | 面向学习、工作、运动等真实场景，提供全屏计时、目标记录、完成结算和计划同步 |
| 打卡奖励 | 完成后展示星光奖励、宝箱进度、生态解锁、成就反馈和跳转入口 |
| 每日提醒 | 按计划设定时间推送通知，点击直达行动舱 |
| 生态区 | 六个维度分别对应独立生态区，按打卡次数解锁建筑、装饰和成长状态 |
| 数据页 | 展示今日进度、周统计、维度分布、最近记录和热力图 |
| 我的 | 支持昵称、星球名、提醒、数据导出、云同步说明和本地数据清理 |

## 技术实现

- **前端**：微信小程序原生框架（WXML、WXSS、JavaScript）
- **组件**：`planet-canvas`、`checkin-card`、`progress-ring`
- **工具层**：日期处理、奖励计算、目标推荐、成就判断、生态解锁、本地存储、云同步、音效反馈
- **数据策略**：本地 Storage 优先，微信云开发异步同步；网络失败时自动降级为纯本地模式
- **云函数**：`login`、`submitCheckin`、`getDashboard`、`exportData`、`updateProfile`、`getAssetUrl`
- **资源**：小程序本地压缩资源 + 微信云存储大图/视频资源
- **测试**：Node.js Test Runner，覆盖日期工具、奖励计算、目标推荐、生态解锁、成就判断、数据统计、计划存储和本地数据清理等核心逻辑

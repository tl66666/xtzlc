# 星途自律舱

星途自律舱是一款星球养成式自律打卡微信小程序。项目把运动、饮食、学习、工作、计划、睡眠六类日常行动，转化为星球生态成长、星光奖励、宝箱和成就徽章，让打卡不只是“点一下完成”，而是有目标、有反馈、有成长感的持续体验。

项目展示页：<https://tl66666.github.io/xtzlc/>

源码目录：[`星途自律舱`](./星途自律舱)

## 核心功能

- 星球启动：首次进入先启动星球，再命名并选择阶段目标。
- 目标系统：围绕均衡自律、学习提升、健身塑形、项目冲刺等节奏生成推荐行动。
- 自定义计划：用户可以自己添加计划，选择六类打卡维度，保存为长期行动。
- 六维打卡：运动、饮食、学习、工作、计划、睡眠分别对应不同生态区。
- 即时反馈：打卡后弹出星光结算，提示本次点亮的生态变化。
- 完成奖励：当日六项完成后可领取宝箱，获得额外星光和成就进度。
- 星球生态：打卡次数会推动荒芜星球逐步生长，并解锁生态区建筑。
- 成就系统：记录连续打卡、首次点亮、六维完成、星光累计等里程碑。
- 数据统计：提供今日进度、周完成率、维度分布和热力图。
- 云开发：包含登录、打卡提交、统计读取、资料更新、数据导出和云资源链接云函数。

## 技术实现

- 后端：微信云开发、云函数、云数据库、云存储
- 数据：本地优先缓存，云端同步

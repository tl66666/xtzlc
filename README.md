# 星途自律舱

星途自律舱是一款星球养成式自律打卡微信小程序。项目把运动、饮食、学习、工作、计划、睡眠六类日常行动，转化为星球生态成长、星光奖励、宝箱进度和成就徽章，让打卡从“点一下完成”变成有目标、有反馈、有成长感的持续体验。

- 小程序源码目录：[`星途自律舱/`](./星途自律舱)

## 核心功能

| 模块 | 说明 |
| --- | --- |
| 成长星球 | 首页展示星球状态、日期、星光值、今日进度和目标入口。星球会随打卡积累从荒芜逐步复苏。 |
| 六类生态区 | 运动绿洲、饮食花园、学习天文台、工作基地、计划灯塔、月光观测站，每类行动对应独立生态成长。 |
| 目标体系 | 用户先选择近期成长节奏，系统根据目标推荐任务顺序和今日重点。 |
| 计划中心 | 用户可创建、编辑、删除自己的计划，设置归属维度、具体行动、目标量、提醒时间和备注。首页优先展示下一项计划，可直接进入行动舱。 |
| 行动舱 | 学习、工作、运动等行动进入沉浸计时场景，实时显示投入时长、目标进度、开始时间，完成后弹出结算页。 |
| 即时奖励 | 完成行动后立即展示星光、星光币、生态解锁、宝箱进度和新成就，不需要用户滑动寻找反馈。 |
| 成就与数据 | 支持成就墙、热力图、趋势数据、连续天数、本周完成率和数据导出。 |
| 微信云开发 | 包含登录、打卡提交、统计读取、资料更新、数据导出和云资源链接等云函数目录。 |

## 计划与行动闭环

1. 用户在计划中心创建每日计划，例如“20:00 刷算法题 2 道”。
2. 计划卡片会显示提醒时间、归属生态区和今日状态。
3. 用户点击“进入行动舱”，开始计时并记录实际产出。
4. 完成后弹出星光结算，计划自动标记为今日完成。
5. 星光和生态成长同步到首页、生态区、数据页和成就墙。

## 技术实现

- 前端：微信小程序原生框架（WXML、WXSS、JavaScript）
- 视觉反馈：Canvas 2D、云存储图片资源、音效反馈
- 数据：本地 Storage 优先，微信云开发同步
- 云函数：`login`、`submitCheckin`、`getDashboard`、`exportData`、`updateProfile`、`getAssetUrl`
- 测试：Node.js Test Runner

# 星途自律舱

**星球养成式自律打卡微信小程序** — 把运动、饮食、学习、工作、计划、睡眠六类日常行动，转化为星球生态成长、星光奖励、宝箱和成就徽章。让打卡不只是"点一下完成"，而是有目标、有反馈、有成长感的持续体验。

> 🔗 项目展示页：<https://tl66666.github.io/xtzlc/>  
> 📁 小程序源码：[`星途自律舱/`](./星途自律舱)

| 📊 **数据复盘** | 热力图、趋势折线、星光值、连续天数、本周完成率 |
| 🚀 **新手引导** | 星球启动动画 + 命名 + 目标选择，建立归属感 |
| 🔁 **云端同步** | 微信云开发：登录、打卡提交、统计读取、资料更新、数据导出、云资源链接 |
| 🧪 **测试覆盖** | 7 个测试文件，覆盖日期、目标、奖励、成就、统计、生态、动机 |

## 六大生态区

每个维度独立成长，4 级解锁（1 次 / 3 次 / 7 次 / 14 次打卡）：

| 生态区 | 维度 | 场景名 | 解锁路径 |
|--------|------|--------|---------|
| 🏃 运动绿洲 | 运动 | 跑道、球场和围栏花园 | 起点跑道 → 晨跑花径 → 能量看台 → 训练营地 |
| 🥗 饮食花园 | 饮食 | 果园小屋和露台餐桌 | 补水井 → 果树篱笆 → 露台餐桌 → 丰收小屋 |
| 📚 学习天文台 | 学习 | 图书馆和观测穹顶 | 阅读灯 → 书页花园 → 图书馆侧楼 → 观测穹顶 |
| ⚡ 工作基地 | 工作 | 办公楼、实验室和信号塔 | 启动工位 → 任务路径 → 实验室 → 信号塔 |
| 🎯 计划灯塔 | 计划 | 灯塔、航线牌和任务广场 | 方向牌 → 任务广场 → 灯塔光束 → 航线中心 |
| 🌙 月光观测站 | 睡眠 | 望远镜、月光花园和静眠舱 | 月光长椅 → 静眠花圃 → 观测台 → 梦境圆顶 |

## 奖励计算规则

- 基础打卡：10 星光值
- 努力加权（轻量/标准/深度）：+0 / +4 / +10
- 记录笔记（6 字以上）：+2
- 完美一天（六项全完成）：+30
- 连续打卡 7 天倍数日：+80

## 打卡动作卡

每维度提供轻量/标准/深度三档：

| 维度 | 轻量 | 标准 | 深度 |
|------|------|------|------|
| 运动 | 微汗启动 15分钟 | 绿洲建设 30分钟 | 突破训练 45分钟+ |
| 饮食 | 补水果园（饮水） | 三餐巡检（健康评分） | 健康餐盘（全天规律） |
| 学习 | 专注 25分钟 | 知识推进 60分钟 | 深度沉浸 90分钟+ |
| 工作 | 清掉一个小任务 | 项目推进 1小时 | 深度工作舱 2小时 |
| 计划 | 写下三件事 | 任务复盘 | 航线规划（周计划） |
| 睡眠 | 睡前降噪 | 规律作息 | 早睡航线 |

**12 个预热模板**：快速拉伸、轻松跑步、饮水 8 杯、三餐规律、编程 60 分钟、阅读 30 页、番茄钟 1 轮、项目推进、今日三件事、晚间复盘、早睡目标、睡前远离手机。

## 技术实现

| 层面 | 技术 |
|------|------|
| 前端 | 微信小程序原生（WXML + WXSS + JavaScript）+ Canvas 2D |
| 后端 | 微信云开发（云函数 + 云数据库 + 云存储） |
| 数据 | 本地 Storage 优先 + 云端同步 |
| 测试 | Node.js Test Runner（7 个测试文件） |
| 部署 | 微信开发者工具上传 + 云存储 CDN |

**云函数清单**：`login`（登录）、`submitCheckin`（打卡提交）、`getDashboard`（统计读取）、`exportData`（数据导出）、`updateProfile`（资料更新）、`getAssetUrl`（云资源链接）。

## 关于

**星途自律舱** 的核心理念是 **"先让用户有目标，再让打卡有反馈"**。用户不是在打卡清单上打钩，而是在通过目标、计划、行动舱、奖励结算、生态成长和数据复盘，持续建设一颗会呼吸、会生长的星球。

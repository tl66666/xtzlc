# 星途自律舱

星途自律舱是一款 **星球养成式自律打卡微信小程序**。项目把运动、饮食、学习、工作、计划、睡眠六类日常行动，转化为星光奖励、生态区建设、宝箱进度和成就徽章，让打卡不只是“点一下完成”，而是形成有目标、有反馈、有成长感的持续体验。

## 项目亮点

- **星球成长叙事**：用户首次进入完成启动动画和星球命名，之后每次行动都会让星球从荒芜逐步复苏。
- **六类生态区**：运动绿洲、饮食花园、学习天文台、工作基地、计划灯塔、月光观测站，对应六类习惯成长。
- **计划与行动舱**：用户可创建、编辑、删除自己的计划，设置归属维度、具体行动、目标量和提醒时间；执行时进入行动舱计时并记录成果。
- **即时奖励反馈**：打卡完成后直接弹出星光、星光币、宝箱进度、生态解锁和新成就，避免用户完成后没有感知。
- **数据可控**：昵称、星球名、目标、计划、提醒、本地数据均提供编辑或清理入口，用户可以重新定制自己的使用节奏。
- **数据复盘**：包含连续天数、本周完成率、热力图、维度趋势、成就墙和数据导出。

## 核心流程

```text
启动星球
  ↓
命名星球
  ↓
选择近期目标
  ↓
创建每日计划 / 进入行动舱
  ↓
完成打卡并结算奖励
  ↓
生态区成长、成就解锁、数据沉淀
```

## 功能模块

| 模块 | 说明 |
| --- | --- |
| 星球总览 | 首页展示星球名称、日期、星光、今日进度、目标状态、生态入口和下一步行动。 |
| 目标体系 | 支持选择和清除近期目标，目标会影响首页推荐、打卡模板和行动优先级。 |
| 计划中心 | 支持创建、编辑、删除计划，设置类别、行动内容、目标量、提醒时间和备注。 |
| 行动舱 | 面向学习、工作、运动等真实场景，提供计时、目标记录、完成结算和计划同步。 |
| 打卡奖励 | 完成后展示星光奖励、宝箱进度、生态解锁、成就反馈和跳转入口。 |
| 生态区 | 六个维度分别对应独立生态区，按打卡次数解锁建筑、装饰和成长状态。 |
| 数据页 | 展示今日进度、周统计、维度分布、最近记录和热力图。 |
| 我的 | 支持昵称、星球名、提醒、数据导出、云同步说明和本地数据清理。 |

## 技术实现

- 前端：微信小程序原生框架（WXML、WXSS、JavaScript）
- 组件：`planet-canvas`、`checkin-card`、`progress-ring`
- 工具层：日期处理、奖励计算、目标推荐、成就判断、生态解锁、本地存储、云同步、音效反馈
- 数据策略：本地 Storage 优先，微信云开发同步
- 云函数：`login`、`submitCheckin`、`getDashboard`、`exportData`、`updateProfile`、`getAssetUrl`
- 资源：小程序本地压缩资源 + 微信云存储大图/视频资源
- 测试：Node.js Test Runner

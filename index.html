Hello! I'm your planet

从荒芜到繁荣，由每次行动点亮。

项目不是单纯的打卡清单，而是围绕「目标设定、计划生成、当日行动、即时奖励、生态变化、数据复盘」构建的成长闭环。 用户完成一次行动，就会获得明确反馈：获得多少星光、点亮哪个区域、距离宝箱还有多远、下一步该做什么。

Feature system

不是点一下完成，而是一次完整的成长反馈。

01

目标与计划

先选择成长节奏，再生成六类行动计划；也支持用户自己添加运动、饮食、学习等日常安排。

02

六维打卡

每类打卡对应一个生态区，不同习惯会推动不同建筑、装饰和光效出现。

03

奖励弹窗

完成后即时展示星光、宝箱进度和解锁变化，让用户不需要滑回顶部寻找反馈。

04

宝箱与成就

六项完成领取宝箱；连续打卡、首次点亮、完美一天会沉淀为成就徽章墙。

05

星球生态

首页是总览星球，点击进入六个生态区，查看建筑群、任务状态和成长记录。

06

数据复盘

通过热力图、趋势、连续天数和星光值，让自律不只停留在当天完成。

Go beyond

完成行动后，立刻看见奖励。

打卡反馈采用「星光结算 + 生态变化 + 宝箱进度 + 成就提示」组合，让用户知道自己不是只点了一个按钮， 而是真的推动星球向下一阶段生长。

Implementation

微信小程序 + 云开发 + 本地优先数据流。

原生小程序JavaScript / WXML / WXSS 构建核心页面和交互。
微信云开发云函数处理登录、打卡提交、统计读取、资料更新和数据导出。
本地优先本地 Storage 保证即时反馈，云端同步用于备份和恢复。
资源分层大图和视频走云存储，主包保留压缩后的必要资源。

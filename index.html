功能体系

完整的自律反馈系统

每一次行动都有明确反馈，让用户知道自己不是只点了一个按钮，而是真的在推动星球生长。

目标与计划

先选择成长节奏，再生成六类行动计划；支持自定义添加运动、饮食、学习等日常安排。

六维打卡

每类打卡对应一个生态区，不同习惯会推动不同建筑、装饰和光效在星球上出现。

即时奖励

完成后即时展示星光值、宝箱进度和解锁变化，反馈就在当前页面，无需寻找。

宝箱与成就

六项完成领取宝箱；连续打卡、首次点亮、完美一天会沉淀为成就徽章墙。

星球生态

首页是总览星球，点击进入六个生态区，查看建筑群、任务状态和成长记录。

数据复盘

通过热力图、趋势曲线、连续天数和星光值统计，让自律不只停留在当天完成。

打卡提醒舱

为每个生态区独立设置提醒时间，微信订阅消息准时推送，不再错过任何一次成长机会。支持自定义提醒文案和重复周期。

心情日志

每次打卡可记录当下心情与备注，长期追踪情绪与自律的关联，发现最适合自己的成长节奏。

六大生态区

每个习惯都有专属领地

六类日常行动对应六个独特生态区，打卡越多，生态越繁荣。

技术实现

微信小程序 + 云开发

零服务器成本，完整前后端架构，本地优先的数据流设计。

原生小程序 JavaScript / WXML / WXSS 构建核心页面与交互动画，Canvas 绘制星球与星空背景。
微信云开发 云函数处理登录、打卡提交、统计读取、资料更新和数据导出，零服务器运维成本。
本地优先 本地 Storage 保证即时反馈，云端同步用于备份和恢复，弱网环境下仍可正常使用。
资源分层 大图和视频走云存储，主包保留压缩后的必要资源，控制小程序体积在 2MB 以内。

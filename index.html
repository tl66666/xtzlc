核心亮点

三大核心机制，让自律真正发生

专注模式

制定计划后进入行动舱，全屏沉浸计时。计时期间屏蔽干扰，结束后自动结算星光并同步计划完成度。让每一次专注都有起止、有成果。

计划打卡

创建每日计划，设置维度、目标量和提醒时间。完成后自动归档并计入星球成长，未完成的计划次日提醒，形成完整执行链路。

每日提醒

每个计划独立设置提醒时间，到点微信推送通知，点击直达行动舱。连续打卡天数实时显示在首页，断签清零，激励持续坚持。

技术实现

原生微信小程序 + 云开发

使用 WXML、WXSS、JavaScript 和微信云开发完成数据同步、云函数、云存储与本地缓存，兼顾演示完整度和真实可用性。

JS
原生小程序框架WXML + WXSS + JS 构建9个页面，3个自定义组件（planet-canvas / checkin-card / progress-ring），模块化工具层（日期/存储/生态系统/奖励系统等）
云
微信云函数6个云函数：login登录、submitCheckin打卡提交、getDashboard数据看板、exportData导出、updateProfile资料更新、getAssetUrl资源获取
DB
云数据库 + 本地优先本地 Storage 即时读写反馈，云端 MongoDB 异步同步；网络失败时自动降级为纯本地模式，数据不丢失
2D
Canvas + 云存储 + 音频Canvas 2D 绘制星球纹理与生态指示器；星球贴图、生态图、引导图等静态资源托管于云存储 CDN；打卡成功/宝箱解锁触发音效反馈
